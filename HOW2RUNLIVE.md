HOW TO SET UP & START

1. Open git bash

2. Navigate to the project folder (e.g cd downloads/) & if needed more times then cd .. cd ..

3. Install yarn (if needed)
>  yarn install

4. Navigate to > frontend in bash
>  use /cd  >>> /test-task/frontend

4. Start server
>  yarn dev

5. Open server 
>  CTRL + CLICK (URL) 
or
http://localhost:5173
